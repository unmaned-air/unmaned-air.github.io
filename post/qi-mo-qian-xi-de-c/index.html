以下是我在复习的过程中发现的华点

一些容易被搞到的知识点

  • 编写C++程序一般需要经过的几个步骤依次是：编辑、编译、连接、运行
  • 赋值兼容性规则是指派生类对象可以当作基类对象使用，只要存在继承关系，所以单继承或多继承都适用。

空指针和野指针

空指针： 指针变量指向内存中编号为0的空间，可以初始化指针变量（因为野指针野一点），但空指针指向的内存是不可以访问的。
野指针： 指针变量指向非法的空间（比如指向地址编号为某某的空间，比如指针指向数组溢出的位置）

const修饰指针

  • 常量的指针（const int *p）
    指针指向可以修改，但指针指向的值不可修改
    理解为const int 后面带了个 * 指针
  • 指针的常量（int * const）
    指针指向不可修改，指针指向的值可以修改
    理解为 * 后面带着const ，const关键字修饰的是指针变量
  • 同时修饰
    指针指向与指针指向的值都不可以修改。
int main() {

	int a = 10;
	int b = 10;

	//const修饰的是指针，指针指向可以改，指针指向的值不可以更改
	const int * p1 = &a; 
	p1 = &b; //正确
	//*p1 = 100;  报错
	

	//const修饰的是常量，指针指向不可以改，指针指向的值可以更改
	int * const p2 = &a;
	//p2 = &b; //错误
	*p2 = 100; //正确

    //const既修饰指针又修饰常量
	const int * const p3 = &a;
	//p3 = &b; //错误
	//*p3 = 100; //错误

	system("pause");

	return 0;
}

内存分区模型

  1. 代码区：就是代码所占的内存；
  2. 全局区：存放全局变量和静态变量；
  3. 栈区：由编译器自动分配释放，存放函数的参数值、局部变量等；
  4. 堆区：由程序员分配和释放，若程序员不释放，程序结束时由操作系统回收。（new）

不同区域存放的数据赋予不同的生命周期，给我们更大的灵活编程

new操作符

C++中利用new操作符在堆区开辟内存，在程序的最后要记得 delete 数据。
利用new创建的数据，会返回该数据对应类型的指针。
需要注意的是开辟数组的时候的操作是层层覆盖的。

  • 示例1
int* func()
{
	int* a = new int(10);
	return a;
}
int main() {
	int *p = func();
	cout << *p << endl;
	//利用delete释放堆区数据
	delete p;
	//cout << *p << endl; //报错，释放的空间不可访问
	return 0;
}
  • 示例2
//堆区开辟数组
int main() {
	int* arr = new int[10];
	for (int i = 0; i < 10; i++)
	{
		arr[i] = i + 100;
	}
	for (int i = 0; i < 10; i++)
	{
		cout << arr[i] << endl;
	}
	//释放数组 delete 后加 []
	delete[] arr;
	return 0;
}
  • 示例3
//动态内存分配
//new开辟二维数组空间
#include<iostream>
#include<algorithm>
using namespace std;

void chan(int **a)
{
	int b[3][3];
	for (int i = 0; i < 3; ++i)
		for (int j = 0; j < 3; ++j)
			b[i][j] = a[j][i];
	for (int i = 0; i < 3; ++i)
		for (int j = 0; j < 3; ++j)
			a[i][j] = b[i][j];
}

int main()
{
	int **a;
	a = new int*[3];
	for (int i = 0; i < 3; ++i)
		a[i] = new int[3];
	int k = 0;

	for (int i = 0; i < 3; ++i)
		for (int j = 0; j < 3; ++j)
			a[i][j] = k++;

	chan(a);
	for (int i = 0; i < 3; ++i)
	{
		for (int j = 0; j < 3; ++j)
			cout << a[i][j] << ' ';
		cout << endl;
	}
	for (int i = 0; i < 3; ++i)
		delete[] a[i];
	delete[] a;
	return 0;
}

引用

注意事项
  • 引用必须初始化。

引用变量是一个别名，也就是说它是某个已存在变量的另一个名字。一旦把引用初始化为某个变量，就可以使用该引用名称或变量名称来指向变量。

  • 引用在初始化之后不可以改变。
引用和指针的不同之处
  • 不存在空引用。引用必须连接到一块合法的内存。
  • 一旦引用被初始化一个对象，就不能被指向到另一个对象。指针可以再任何时候指向到另一个对象。但是指针可以在 任何时间 指向到另一个对象。
  • 引用必须在创建时被初始化。指针可以在任何时间被初始化
把引用作为参数值与返回值
C中有利用&达到传入数字进行swap操作，C++中同样有类似操作

#include<stdio.h>
void swap(int* a, int* b)
{
	int temp = *a;
	*a = *b;
	*b = temp;
}
int main()
{
	int a = 10;
	int b = 20;
	swap(&a, &b);
	printf("%d,%d", a, b);
	return 0;
}
  • C++中把引用作为参数
#include <iostream>
using namespace std;
void swap(int& x, int& y);
int main ()
{
   int a = 100;
   int b = 200;
   cout << "交换前，a 的值：" << a << endl;
   cout << "交换前，b 的值：" << b << endl;
   swap(a, b);//最突出的一点是a,b不带任何符号，但却能进行换位
   cout << "交换后，a 的值：" << a << endl;
   cout << "交换后，b 的值：" << b << endl;
 
   return 0;
}
void swap(int& x, int& y)
{
   int temp;
   temp = x; /* 保存地址 x 的值 */
   x = y;    /* 把 y 赋值给 x */
   y = temp; /* 把 x 赋值给 y  */
  
   return;
}
